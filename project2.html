Used Terraform to deploy a remote development environment

  • Source code

Written a Terraform Script to deployed a simulated remote development environment in AWS which runs a EC2 instance in a public subnet running docker.

Utilized several terraform tools and functions like terraform state, format, replace, console, variables, conditionals, template file, etc.

Used userdata template and local exec provisioner to bootstrap the EC2 instance and it's connection information to VSCode SSH config file.

    Tools & Service Used:
  • Terraform & Docker
  • VPC, EC2, S3, Route Tables, Subnets, Internet Gateway, EBS
